## Installation

The bundle is not yet on packagist make sure to add the following to your `composer.json` file:

```json
{
    "url": "https://github.com/2lenet/FileBundle",
    "type": "git"
}
```

Install with composer:

```shell
composer require 2lenet/file-bundle
```
